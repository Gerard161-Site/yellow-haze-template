# Yellow Haze Template

A warm, vibrant golden amber template with retro-funk energy. Bold typography meets sun-drenched haze aesthetics for brands that want to stand out with warmth and personality.

## Installation

This template can be uploaded to BudStack via the Template Management interface.

### Upload to BudStack

1. **Push this repository to GitHub** (see commands below)
2. Go to **BudStack Super Admin → Store Templates**
3. Click **"Upload New Template"**
4. Fill in the form:
   - **Template Name:** "Yellow Haze"
   - **GitHub URL:** `https://github.com/{your-username}/yellow-haze-template.git`
   - **Structure Type:** "Default (BudStack)"
5. Click **Upload**
